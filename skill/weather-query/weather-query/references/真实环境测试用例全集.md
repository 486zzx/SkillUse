# 天气查询 Skill 真实环境测试用例全集

本文档定义对 `run_weather_search.py` 在**真实环境**（真实心知天气 API、已配置 Key）下的完整测试用例集，用于回归与验收。

---

## 一、测试范围与前提

| 项目 | 说明 |
|------|------|
| 被测对象 | `scripts/run_weather_search.py` |
| 环境 | 真实心知天气 API，已配置 SENIVERSE_KEY（或脚本内默认 Key） |
| 执行方式 | `python scripts/run_weather_search.py <位置参数...>` |
| 通过标准 | 成功用例：exit=0，输出为合法 JSON，`success=true`，`result` 含 location/current/daily；失败用例：exit≠0 或 success=false，error 含明确提示 |

## 三、执行方式

### 3.1 单用例执行示例

```bash
cd .cursor/skills/weather-query
python scripts/run_weather_search.py 北京
python scripts/run_weather_search.py 上海 明天 --days 5
python scripts/run_weather_search.py 北京 --with-suggestion --with-air
python scripts/run_weather_search.py
```

### 3.2 批量执行（自动化脚本）

```bash
cd .cursor/skills/weather-query
python scripts/run_real_env_tests.py
```

执行后将生成 `references/real_env_test_report.json`，并可在控制台看到每用例的 PASS/FAIL。

## 四、通过标准汇总

- **成功类用例（TC-01～TC-14、TC-16）**：进程 exit code=0；标准输出为单行 JSON；`success=true`；`result` 中必有 `location`（含 name）、`current`、`daily`（数组）；若带 `--with-suggestion` 则 `result.suggestion` 存在；若带 `--with-air` 则 `result.air` 存在。
- **失败类用例（TC-15）**：进程 exit code≠0 或输出中 `success=false`；`error` 字段包含对缺少地点/参数的明确提示。
